# Contribuindo com noovichat-resources

Obrigado pelo interesse! Este repositório aceita contribuições da comunidade
para tornar o ecossistema NooviChat melhor.

## Tipos de contribuição bem-vindos

- ✅ **Tutoriais** em português brasileiro (especialmente passo-a-passo de uso)
- ✅ **Workflows n8n** para casos de uso reais
- ✅ **Snippets de API** em qualquer linguagem
- ✅ **Configurações Docker Compose** para cenários específicos
- ✅ **Correções de docs existentes** (typos, links quebrados, exemplos incorretos)
- ✅ **Adições à awesome-list** (`awesome-noovichat.md`)
- ✅ **Traduções** dos docs em PT para EN/ES (vai trazer audiência internacional)

## Como abrir um Pull Request

1. Fork este repositório
2. Crie uma branch a partir de `main`: `feat/meu-tutorial` ou `fix/typo-readme`
3. Faça suas alterações com commits convencionais:
   - `feat(docs): add tutorial X`
   - `fix(examples): typo in workflow Y`
   - `docs(awesome): add Z to list`
4. Abra o PR descrevendo:
   - O que mudou
   - Por que essa mudança
   - Como testou (se aplicável)

## Política de bugs / problemas no produto

Este repositório é só de **recursos públicos** — issues sobre bugs no produto
NooviChat (instalação, dashboard, API com erro, etc) **não devem ser abertas
aqui**. Use:

- **Roadmap público com votação**: https://noovichat.com/roadmap
- **E-mail de suporte**: contato@noovichat.com
- **Cliente pago**: WhatsApp do time Noovi (consulte seu onboarding)

## O que NÃO aceitar via PR

- ❌ Código que tenta engenharia reversa do produto
- ❌ Credenciais reais (API keys, license keys, secrets)
- ❌ Domínios de clientes específicos sem autorização escrita
- ❌ Detalhes que possam revelar infraestrutura interna da Noovi
- ❌ Modificações da licença MIT deste repo

## Estilo de markdown

- Português brasileiro como idioma principal
- Code blocks com linguagem especificada (` ```bash `, ` ```javascript `, etc)
- Links relativos para outros docs deste repo (não absolute URLs)
- Headings em sentence case (não Title Case): "Como instalar" não "Como Instalar"

## Code of Conduct

Esperamos comportamento respeitoso e profissional. Discriminação, assédio ou
linguagem ofensiva não são tolerados. Comportamento inadequado pode resultar
em ban da comunidade.

---

Em caso de dúvida sobre qualquer ponto: abra uma **Discussion** no repo (não
issue) e marcamos para responder.
